# youtube-party
Watch youtube together with your friends

# Usage
Run the application both as a server and a client (it uses 3000 port and 3001 port).
```
npm start
```

Run all the test suites.
```
npm run test
```

Run the static analysis (Using eslint)
```
npm run lint
npm run fix-lint
```

Format your code
```
npm run format
```
